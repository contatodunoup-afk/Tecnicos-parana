# Mapa Prática Service+ Paraná PR

## Descrição
Mapa interativo com localização dos técnicos de assistência da Prática Service+ no estado do Paraná, desenvolvido com Leaflet.js.

## Características
- 🗺️ Mapa interativo com Leaflet.js
- 📍 13 localizações técnicas no Paraná
- 📱 Design responsivo
- 🌟 Identificação visual de empresas Service Plus+ com logotipo exclusivo
- 🔍 Zoom adaptativo com redimensionamento automático de marcadores
- 📋 Popups informativos com dados de contato

## Empresas Service Plus+
Marcadas com o logotipo da Nova Identidade Serviços:.
- ELC Manutenção
- JAT Assistência Tec.
- Assistência Técnica Corretiva
- WMA Assistência Técnica LTDA
- LUBRAS Soluções e Serviços LTDA
- SOS Fogões
- GRAAL Refrigeração

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)
- Leaflet.js 1.9.4
- OpenStreetMap

## Como usar
1. Abra `mapa.html` no navegador
2. Clique nos marcadores para visualizar informações das empresas
3. Use o zoom para ampliar/reduzir a visualização

## Estrutura de Arquivos
- `mapa.html` - Arquivo principal do mapa
- `data_fundepar.json` - Dados das empresas (incluindo classificação)
- `logo.jpg` - Logo Prática Service+
- `NovaIdentidade_Servicos_v03_20250910-02.jpg` - Logotipo para Service Plus+
- `Giovane Oliveira.jpeg` - Foto do técnico próprio
